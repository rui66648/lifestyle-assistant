# 【生活娱乐】生活习惯小助手 — 古今结合的AI科学养生习惯追踪应用

## 1. Demo 简介

**是什么：** 一款**古今结合**的 PWA 养生习惯追踪应用。以《黄帝内经》等14部经典著作为理论基础，以现代行为科学和健康管理方法为实践框架，将千年中医智慧转化为可量化、可追踪、可执行的每日健康计划。

**面向谁：**
- 想了解中医养生但觉得文言文门槛太高的年轻人
- 追求**科学养生**、希望养生建议有据可查的理性用户
- 需要系统化健康习惯追踪工具的都市人群

**核心理念：古今结合，科学养生**

每一条健康建议都遵循"**古籍原文 → 现代解读 → 可执行方案**"的三层结构。用户不仅知道"该怎么做"，更知道"为什么这么做的依据是什么"。点击任意出处，即可跳转14部参考文献的完整原文页面。

**主要功能：**

- **📅 今日养生卡片**：公历+农历双日期、季节养生建议、24节气Badge、随机养生名言。名言均来自14部经典，点击出处直达原文。
- **💧 智能饮水追踪**：8杯可视化+毫升双单位、时间线计划、2小时未喝提醒、周趋势分析。把"食饮有节"的古训转化为现代饮水科学。
- **💭 情志养生记录**：记录怒喜思悲恐，对照五脏-情志关系，提供《素问》情志相胜调节法。古籍智慧+现代情绪管理。
- **🛡️ 五劳防护指南**：久视伤血→闭目养肝、久坐伤肉→起身活动。把《素问·宣明五气篇》的五劳理论对应到办公室、手机族等现代场景。
- **⏳ 子午流注时辰**：12时辰×12经络，每个时辰告诉你"此时哪个脏腑当令、该做什么"。古代时间医学的现代应用。
- **📦 黄帝内经习惯包**：四季养生+五色饮食+情志调理+五劳防护，7个子包34个习惯，零基础系统入门中医养生。
- **📚 参考文献文库**：14部经典著作的数字化阅读入口，从先秦到现代，从《黄帝内经》到《科学休息》。

## 2. Demo 创作思路

**灵感来源：**

市面上养生APP很多，但大多是"告诉你该做什么"，不解释"为什么"。中医养生被贴上"玄学"标签，很大程度上是因为普通人读不懂古籍原文，只能听二手解读。

另一方面，纯西医视角的健康APP又缺乏整体观和个性化调理思维。现代人需要的不是"中医vs西医"的二选一，而是**古今智慧的融合**。

**想解决的问题：**

1. **养生知识碎片化**：小红书上碎片化的养生tips缺乏系统性和理论根基
2. **古籍门槛太高**：《黄帝内经》等经典是文言文，普通人望而却步
3. **"伪科学"泛滥**：很多养生建议没有文献出处，用户无法辨别真伪
4. **习惯工具缺乏深度**：普通打卡APP只是记录，不提供健康指导

**为什么做这个方向：**

中医不是玄学，而是一门**有数千年临床实践积累的医学体系**。《黄帝内经》成书于先秦，其中的"法于阴阳，和于术数，食饮有节，起居有常"至今被现代医学研究反复验证。

这个项目的核心判断是：**用AI降低古籍阅读门槛，用现代科技手段验证和传播传统养生智慧**。每一条建议都标注古籍原文出处，用户可以随时溯源查证。这不是"信则灵"，而是"有据可查的科学养生"。

## 3. 古今养生知识科普

应用内整合的14部参考文献，跨越两千余年，形成一条完整的**古今养生知识谱系**：

### 经典古籍（理论奠基）

| 著作 | 年代 | 核心贡献 |
|:---|:---|:---|
| 《黄帝内经》 | 先秦 | 中医养生理论之源，阴阳五行、脏腑经络、四时调神 |
| 《养生论》 | 三国 | 首创系统养生专论，提出"形神兼养" |
| 《抱朴子》 | 晋代 | 道家养生，"不伤为本"的养生总则 |
| 《备急千金要方》 | 唐代 | 药王孙思邈的"养性"篇，食疗与导引并重 |
| 《饮膳正要》 | 元代 | 宫廷营养学专著，第一部营养学专著 |
| 《遵生八笺》 | 明代 | 古代养生百科全书，八笺涵盖养生各方面 |
| 《老老恒言》 | 清代 | 老年养生专著，寝居、饮食、导引的实用指南 |
| 《闲情偶寄》 | 清代 | 生活美学与养生，"养生先养心" |

### 现代著作（科学验证）

| 著作 | 作者 | 核心贡献 |
|:---|:---|:---|
| 《你是你吃出来的》 | 夏萌 | 细胞营养学，七大营养素的现代解读 |
| 《九种体质养生全书》 | 王琦 | 体质辨识与个性化调理 |
| 《科学休息》 | 亚历克斯 | 注意力网络理论，四种休息方式的科学依据 |
| 《求医不如求己》 | 中里巴人 | 穴位按摩与经络保健的实用指南 |
| 《拉伸》 | 鲍勃·安德森 | 拉伸动作的科学原理与办公室应用 |

**古今对照示例：**

> **古籍原文：** "久视伤血，久卧伤气，久坐伤肉，久立伤骨，久行伤筋。" ——《素问·宣明五气篇》
>
> **现代解读：** 长时间保持同一姿势会导致局部血液循环不畅、肌肉僵硬、关节压力增大。现代医学研究证实，每坐45分钟起身活动5分钟可显著降低腰椎疾病和代谢综合征风险。
>
> **可执行方案：** 应用内设置"少坐多动"习惯，每45分钟提醒一次，记录起身活动次数。

## 6. 额外分享：开发心得

**最大的收获：** AI 让"一个人做完整产品"成为可能。更意外的是，AI 在跨学科知识整合上表现出色——把《黄帝内经》的"怒伤肝"与现代心理学的情绪管理理论进行对照分析，生成既忠于原文又符合现代认知的健康建议。

**古今结合的难点：** 最大的挑战不是技术，而是**内容准确性**。每条健康建议都需要查证古籍原文出处，确保不曲解、不夸大。AI 在这个环节帮助很大：它能快速定位相关章节，提供多版本对照，还能指出某些民间说法在古籍中并无依据。

**下一步计划：**
- 加入中医体质辨识（九种体质测试）
- 个人健康数据趋势预测
- 最终目标：打造**有据可查、古今融合**的 AI 中医健康助手
