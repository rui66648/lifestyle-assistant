# AI 养生顾问部署说明

## 概述

AI 养生顾问功能通过 Cloudflare Workers + 通义千问 API 实现，用户在应用内输入养生问题，AI 基于《黄帝内经》等14部经典给出回答。

## 部署步骤

### 1. 注册阿里百炼获取 API Key

1. 访问 https://bailian.console.aliyun.com
2. 注册/登录阿里云账号
3. 进入"百炼大模型"控制台
4. 点击右上角头像 → API Key 管理
5. 创建新的 API Key，复制保存

> 新用户有免费额度，足够日常使用。

### 2. 注册 Cloudflare 账号

1. 访问 https://dash.cloudflare.com
2. 注册账号（可用邮箱直接注册）
3. 进入 Workers & Pages 控制台

### 3. 创建 Worker

1. 点击"创建服务"或"Create a Service"
2. 服务名称填写 `ai-proxy`（或任意名称）
3. 点击"创建服务"

### 4. 编辑 Worker 代码

1. 点击"快速编辑"或"Quick Edit"
2. 删除默认代码，粘贴 `ai-proxy.js` 中的全部内容
3. 点击"保存并部署"

### 5. 配置环境变量

1. 返回 Worker 主页，点击"设置" → "变量"
2. 添加环境变量：
   - 变量名：`QWEN_API_KEY`
   - 值：你的阿里百炼 API Key
3. 点击"保存"

### 6. 获取 Worker URL

1. 在 Worker 主页，找到触发器（Triggers）
2. 默认会有一个 `*.workers.dev` 的 URL
3. 复制这个 URL，例如：`https://ai-proxy.yourname.workers.dev`

### 7. 配置前端

1. 打开 `index.html`
2. 找到这行代码：
   ```javascript
   const AI_WORKER_URL = '';
   ```
3. 填入你的 Worker URL：
   ```javascript
   const AI_WORKER_URL = 'https://ai-proxy.yourname.workers.dev';
   ```
4. 重新部署前端代码

### 8. 验证

1. 打开应用，点击右下角的 🤖 浮动按钮
2. 输入"失眠怎么调理？"
3. 如果看到 AI 回复，说明部署成功

## 故障排查

| 问题 | 原因 | 解决 |
|:---|:---|:---|
| 显示配置说明 | AI_WORKER_URL 为空 | 按步骤7配置 |
| 网络错误 | Workers URL 错误 | 检查 URL 是否完整 |
| 服务不可用 | API Key 错误 | 检查环境变量 QWEN_API_KEY |
| 回复为空 | API 额度用完 | 检查阿里百炼控制台额度 |

## 费用说明

- **Cloudflare Workers**：免费计划每天 10 万次请求，完全够用
- **阿里百炼**：新用户有免费额度，超出后按量计费（很便宜）
- **前端**：纯静态托管，无费用
