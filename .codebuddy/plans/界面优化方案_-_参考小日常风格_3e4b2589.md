## 产品概述

生活习惯小助手是一款基于《黄帝内经》等中医经典的养生习惯追踪Web应用。当前界面已具备基础功能，但在视觉设计、交互体验、信息层次等方面还有较大优化空间。

## 核心需求

参考"小日常"及头部习惯打卡App的设计风格，对应用界面进行全面优化，提升视觉精致度、交互流畅性和用户体验。

## 优化目标

1. 视觉升级：更现代温暖的配色体系、更精致的卡片设计
2. 交互优化：更流畅的动画反馈、更直观的操作逻辑
3. 信息层次：更清晰的内容分区、更重要的信息突出
4. 情感化设计：温暖的微交互、成就感的视觉反馈

## 技术栈

- 前端：原生 HTML/CSS/JavaScript（无框架）
- 样式：CSS3 变量、Flexbox、Grid、动画
- 存储：LocalStorage（保持离线优先）

## 实施方案

### 1. 配色体系升级

**当前问题**：配色偏灰、对比度不足、缺少层次

**优化方案**：

- 更新 `:root` CSS变量，采用更温暖现代的配色
- 新增阴影体系（`--shadow-sm`、`--shadow-md`、`--shadow-lg`）
- 统一圆角规范（`--radius-sm: 8px`、`--radius: 14px`、`--radius-lg: 20px`）

**关键修改**：`css/main.css` 第1-2行

### 2. 首页（打卡页）优化

#### 2.1 顶部日期区域

**当前HTML结构**（index.html 第21-38行）：

```html
<div class="mini-header" id="todayCard">
  <div style="display:flex;...">
    <span class="mini-date" id="todayDate"></span>
    <span class="mini-lunar" id="todayLunar"></span>
  </div>
  <div class="mini-quote" id="quoteCard">...</div>
</div>
```

**优化方向**：

- 放大日期数字（48px），添加渐变背景
- 箴言卡片改为白色卡片+阴影，添加装饰引号
- 优化节气/农历显示样式

#### 2.2 习惯卡片重设计

**当前CSS**（css/main.css 第83-102行）：

```css
.habit-card {
  background: transparent;
  padding: 14px 16px;
  border-bottom: 1px solid var(--rule);
}
```

**优化方案**：

- 改为白色卡片+阴影设计
- 添加左侧4px彩色条（未完成绿色，已完成渐变绿）
- 打卡按钮从38px增大到44px，添加阴影
- 添加完成状态动画

#### 2.3 进度环区域

**当前实现**（js/ui/render.js 第196-207行）：

- 使用SVG圆形进度环
- 添加鼓励语显示

**优化方向**：

- 增大进度环到140px
- 优化鼓励语卡片样式
- 添加补签按钮样式优化

### 3. 底部导航栏优化

**当前CSS**（css/main.css 第65-72行）：

```css
.bnav-item { opacity: 0.4; }
.bnav-center { width: 44px; height: 44px; }
```

**优化方案**：

- 添加毛玻璃效果（`backdrop-filter: blur(20px)`）
- 选中态添加顶部指示器条（3px高）
- 中间按钮增大到56px，优化渐变和阴影
- 统一图标和文字样式

### 4. 统计页面优化

**当前实现**（js/ui/render.js 第274-308行）：

- 环形进度 + 统计网格 + 排行列表 + 热力图

**优化方向**：

- 统计卡片统一圆角和阴影
- 排行列表添加序号徽章（金、银、铜）
- 热力图优化颜色梯度

### 5. 动画与微交互

**新增动画**：

- 打卡成功：缩放弹出 + 粒子爆炸
- 页面切换：fadeInUp 渐入
- 连续打卡：火焰图标闪烁动画
- 按钮反馈：按压缩放效果

### 6. 暗黑模式支持

**实现方式**：

```css
@media (prefers-color-scheme: dark) {
  :root {
    --bg: #1A1A1A;
    --bg2: #2A2A2A;
    --ink: #F0F0F0;
    /* ... */
  }
}
```

## 设计风格

采用**温暖极简主义**风格，参考"小日常"App的设计精髓：

- **配色**：温暖米白底 + 渐变绿主题色 + 温暖橙辅助色
- **布局**：清晰的内容分区、充足的留白、统一的圆角
- **交互**：流畅的动画反馈、直观的操作逻辑
- **氛围**：有温度、低负担、专注习惯养成

## 页面规划

### 页面1：打卡页（首页）

**布局结构**：

1. **顶部区域**：日期大字 + 农历/节气 + 渐变背景
2. **箴言卡片**：白色卡片 + 装饰引号 + 阴影
3. **进度环**：居中展示 + 鼓励语
4. **习惯列表**：卡片式布局 + 左侧彩色条 + 圆形打卡按钮

**关键优化点**：

- 日期数字放大到48px，使用主题绿色
- 箴言卡片添加 `"` 装饰，优化排版
- 习惯卡片改为白色背景+阴影，提升层次感
- 打卡按钮添加阴影，完成态使用对勾动画

### 页面2：统计页

**布局结构**：

1. **环形进度**：更大的环形图 + 中心百分比
2. **数据网格**：2x2网格 + 渐变色数字
3. **排行列表**：序号徽章 + 进度条
4. **热力图**：优化颜色梯度
5. **成就徽章**：网格布局

**关键优化点**：

- 统计卡片统一圆角14px + 阴影
- 数字使用渐变色（绿到橙）
- 排行前3名添加金/银/铜徽章

### 页面3：管理页

**布局结构**：

1. **等级卡片**：渐变背景 + 进度条
2. **习惯包列表**：可展开卡片
3. **我的习惯**：列表式管理
4. **功能按钮**：网格布局

**关键优化点**：

- 卡片统一圆角和阴影
- 展开/收起添加旋转动画
- 功能按钮优化间距和样式

### 页面4：AI聊天页（弹窗）

**布局结构**：

1. **消息列表**：气泡式对话
2. **输入区域**：圆角输入框 + 发送按钮

**关键优化点**：

- 消息气泡优化圆角和阴影
- 输入框添加焦点态样式
- 发送按钮使用主题渐变

## 交互设计

1. **打卡动画**：点击按钮 → 缩放弹出 → 粒子爆炸 → 卡片状态更新
2. **页面切换**：fadeInUp 渐入动画（0.3s）
3. **按钮反馈**：按压缩放到0.9倍（0.2s）
4. **卡片hover**：轻微上移 + 阴影加深（桌面端）
5. **连续打卡**：火焰图标闪烁动画

## 响应式设计

- 最大宽度480px居中显示
- 底部导航适配安全区域（`env(safe-area-inset-bottom)`）
- 字体大小使用px单位，确保一致性
