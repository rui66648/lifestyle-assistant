## 产品概述

参考"小日常"App 的交互设计，改版"添加习惯"面板（libraryPanel），使其更符合现代移动端习惯管理应用的交互习惯。

## 核心功能

- 习惯库以网格卡片布局展示（图标+名称，2列网格），替代现有列表式布局
- 顶部增加横向滑动分类标签栏，点击切换分类，替代现有分区标题
- 自定义习惯输入区优化为更紧凑的内联样式
- 添加习惯后有视觉反馈（卡片变灰/显示"已添加"）
- 保留健康生活建议包和四季养生包的快捷添加入口

## 技术栈

- 前端：原生 HTML/CSS/JavaScript（现有项目，不引入新框架）
- 样式：CSS Grid 实现网格布局，CSS 实现分类标签横向滚动

## 实现方案

### 修改文件清单

1. `index.html` — 修改 libraryPanel 的 panel-header，增加分类标签栏 HTML
2. `css/main.css` — 新增网格卡片样式、分类标签栏样式
3. `js/ui/panels.js` — 修改 `renderLibraryPanel()` 函数，改为网格布局渲染；增加分类切换逻辑

### 关键设计决策

- **网格布局**：使用 CSS Grid `grid-template-columns: repeat(2, 1fr)`，每张卡片显示图标+名称+添加按钮
- **分类标签栏**：固定在面板顶部，横向滚动（`overflow-x: auto; white-space: nowrap`），点击切换时高亮当前分类并过滤网格内容
- **添加反馈**：添加后卡片添加 `.added` 类，降低透明度+显示"已添加"覆盖层
- **自定义习惯**：折叠在面板底部，点击"自定义习惯"展开

### 实现细节

#### 1. HTML 结构（index.html）

在 `libraryPanel` 的 `panel-body` 内，通过 JS 动态渲染分类标签栏和内容区。分类标签栏需要固定在顶部，不随内容滚动。

#### 2. CSS 样式（main.css）

新增以下样式类：

- `.lib-tabs` — 分类标签栏容器，横向滚动
- `.lib-tab` — 单个分类标签
- `.lib-tab.active` — 当前选中标签
- `.lib-grid` — 网格容器，`display: grid; grid-template-columns: repeat(2, 1fr); gap: 12px;`
- `.lib-card` — 单个习惯卡片，包含图标、名称、添加按钮
- `.lib-card.added` — 已添加状态样式

#### 3. JS 逻辑（panels.js）

- `renderLibraryPanel()` 改为渲染网格卡片
- 新增 `filterLibraryByCategory(cat)` 函数，切换分类时过滤显示
- 分类标签栏点击时调用 `filterLibraryByCategory`

### 性能考虑

- 分类切换只做 DOM 显示/隐藏，不重新渲染整个列表
- 网格卡片数量有限（每个分类最多10个左右），无需虚拟滚动

## 设计风格

参考"小日常"App 的极简设计风格，采用温暖的自然色调，结合项目的养生主题。

### 布局设计

- **分类标签栏**：面板顶部，横向滑动，当前选中分类有底部指示条
- **习惯网格**：2列网格，每张卡片白色背景、圆角、轻微阴影
- **卡片内容**：图标（大）+ 习惯名称 + 添加按钮（小）
- **已添加状态**：卡片半透明+灰色覆盖层+""已添加""文字

### 交互设计

- 点击分类标签：平滑切换到对应分类，当前标签高亮
- 点击添加按钮：按钮文字变为""已添加""，卡片进入已添加状态
- 自定义习惯：面板底部可展开区域

### 视觉风格

- 整体风格：极简、温暖、自然
- 图标：emoji 原生图标，大小 32px
- 圆角：16px（卡片），12px（按钮）
- 阴影：轻微阴影，不抢夺视觉焦点
