# 养生小助手设计系统规范 V2.0

## 1. 设计原则

| 原则 | 说明 |
|------|------|
| **一致性优于创新** | 相同功能使用相同组件，降低学习成本。打卡按钮在首页、记录页、弹窗中样式必须一致。 |
| **五行五色，养生调性** | 色彩系统植根中医五行（木火土金水），主色取"木"之青绿，寓意生机与自然。 |
| **轻量优先，减少干扰** | 间距紧凑、圆角柔和、阴影克制。养生场景需要"宁静致远"，避免视觉噪音。 |
| **响应式适配，移动优先** | 所有组件以 375px 为基准设计，max-width: 480px 居中，确保各机型体验一致。 |
| **无障碍访问** | 文字对比度 >= 4.5:1，点击区域 >= 44*44px，支持系统字体缩放。 |

## 3. 字体排版

### 3.1 字体家族

```css
--font-sans: -apple-system, BlinkMacSystemFont, 'Segoe UI', 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', sans-serif;
--font-mono: 'GeistMono', monospace;
```

**Fallback 策略**：
1. iOS/macOS：苹方
2. Android：思源黑体 -> 系统 UI 字体
3. Windows：微软雅黑 -> 系统 UI 字体

### 3.2 标题层级

| 级别 | 字号 | 字重 | 行高 | 用途 |
|------|------|------|------|------|
| H1 | `24px` (`--text-3xl`) | 700 | 1.25 | 页面主标题、弹窗标题 |
| H2 | `20px` (`--text-2xl`) | 700 | 1.25 | 卡片标题、区块标题 |
| H3 | `18px` (`--text-xl`) | 600 | 1.3 | 列表项标题、设置项标题 |
| H4 | `16px` (`--text-lg`) | 600 | 1.4 | 小节标题、标签文字 |

### 3.3 正文字号

| 类型 | Token | 字号 | 字重 | 行高 | 用途 |
|------|-------|------|------|------|------|

| 小字 | `--text-sm` | `12px` | 400 | 1.5 | 辅助说明、时间戳 |
| 微字 | `--text-xs` | `11px` | 400 | 1.4 | 标签、极小提示 |

## 7. 交互状态

### 7.1 按钮状态

| 状态 | 背景色 | 边框 | 阴影 | 文字色 |
|------|--------|------|------|--------|
| 默认 | `var(--accent)` | 无 | `--shadow-sm` | `#fff` |
| 悬停 | `var(--accent-600)` | 无 | `--shadow` | `#fff` |
| 按下 | `var(--accent-700)` | 无 | `--shadow-xs` | `#fff` |
| 禁用 | `var(--bg2)` | `1px solid var(--rule)` | 无 | `var(--muted)` |
| 加载 | `var(--accent)` + 动画 | 无 | `--shadow-sm` | `#fff` |

**CSS 实现**：
```css
.ui-btn--primary {
  background: var(--accent);
  color: #fff;
  box-shadow: var(--shadow-sm);
  transition: all 0.2s var(--ease);
}
.ui-btn--primary:hover {
  background: var(--accent-600);
  box-shadow: var(--shadow);
}
.ui-btn--primary:active {
  background: var(--accent-700);
  transform: scale(0.98);
  box-shadow: var(--shadow-xs);
}
.ui-btn--primary:disabled {
  background: var(--bg2);
  color: var(--muted);
  box-shadow: none;
  cursor: not-allowed;
}
```

### 7.2 输入框状态

| 状态 | 背景色 | 边框 | 阴影 | 文字色 |
|------|--------|------|------|--------|
| 默认 | `var(--surface)` | `1px solid var(--border)` | 无 | `var(--ink)` |
| 聚焦 | `var(--surface)` | `2px solid var(--accent)` | `0 0 0 3px var(--accent-light)` | `var(--ink)` |
| 错误 | `var(--danger-light)` | `2px solid var(--danger)` | 无 | `var(--danger)` |
| 禁用 | `var(--bg2)` | `1px solid var(--border-light)` | 无 | `var(--muted)` |

## 8. 设置界面

### 8.1 设置项结构

```
[图标]  设置项标题          [控件]
        描述文字（可选）
```

### 8.2 设置项间距

| 元素 | 规格 |
|------|------|
| 设置项高度 | `56px`（无描述）/ `72px`（有描述） |
| 左侧图标尺寸 | `24px`，右边距 `12px` |
| 标题字号 | `--text-base` (14px)，字重 600 |
| 描述字号 | `--text-sm` (12px)，颜色 `var(--muted)` |
| 右侧控件 | Toggle / 数值 / 箭头，宽度 `48px` |
| 分割线 | `1px solid var(--rule)`，左边距 `44px` |

### 8.3 设置项控件

| 控件类型 | 尺寸 | 样式 |
|----------|------|------|
| Toggle 开关 | `44*24px` | 圆角 12px，激活色 `var(--accent)` |
| 数值显示 | `auto` | 右对齐，字号 `--text-base` |
| 箭头 | `24px` | `>` 符号，颜色 `var(--muted)` |
| 按钮 | `auto` | 文字色 `var(--accent)`，无背景 |

### 8.4 设置分组

| 分组 | 内容 |
|------|------|
| **个人设置** | 体质测试结果、昵称、头像 |
| **提醒设置** | 提醒方式、免打扰时段、午休免打扰 |
| **显示设置** | 深色模式、卡片圆角风格、皮肤主题 |
| **数据管理** | 导出数据、清除数据、云同步 |
| **关于** | 版本号、检查更新、隐私政策 |
